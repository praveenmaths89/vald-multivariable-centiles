# Repository Manifest

This document indexes every artefact in this repository.

## Privacy posture

This repository contains **no raw athlete data**. It ships only the
methodology (code), aggregate cohort-level results, the compiled paper,
and a single per-athlete summary table containing the joint percentile
(no per-metric values). The underlying ForceDecks corpus is held by
PhysioQinesis and IIT Bombay; see `data/README.md` for the access
request procedure.

## Verification (`verification/`)

Independent reproduction by `code/VERIFY_ALL.R`, kept as a cross-check
of the paper pipeline.

| Path | Description |
|---|---|
| `verification/numbers/numbers.tex`, `numbers_athlete.tex` | Headline numbers from the verification pipeline |
| `verification/tables/` (6 files) | Cohort-level LaTeX tables |
| `verification/figures/` (7 JPGs) | Cohort-level figures: EDA + one representative radar |
| `verification/results/athlete_scores_summary.csv` | Per-cohort aggregates (means/medians of joint percentile) |
| `verification/results/athlete_joint_scores.csv` | **One row per athlete-test occasion: `athlete_id` (UUID), `sex`, `sport`, `test_type`, `age_years`, `joint_pct`, [`joint_lo`, `joint_hi`,] `joint_band`, `joint_model`, `n_metrics_used`. Per-metric values are NOT exposed.** |

Per-athlete radar plots (one per athlete-test occasion) are NOT
committed. They can be regenerated by running `code/VERIFY_ALL.R`
against the raw data; output lands under
`verification/figures/athletes/<sex>/<sport>/<test>/<athlete_id>__<date>.png`.

## Reproducing the analysis

Granted you have the raw `data/analysis_dataset.csv`:

```
Rscript code/anonymise_for_github.R    # strips name/dob for further sharing
Rscript code/RUN_PAPER_FINAL.R         # fits all marginals + joint copulas
Rscript code/MAKE_PAPER_ASSETS.R       # regenerates paper/figures, paper/tables, paper/numbers.tex
Rscript code/VERIFY_ALL.R              # independent reproduction + per-athlete radars
Rscript code/99_check_paper_assets.R   # lints \input/\includegraphics references
cd paper && latexmk -pdf main.tex      # rebuilds main.pdf
```
